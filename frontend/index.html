✦ Features

Everything You Need to Ship

A production-grade DevOps project with enterprise-level features

Full-Stack Architecture

Node.js + Express backend, MongoDB with Mongoose ODM, JWT authentication, and a beautiful Vanilla JS frontend.

Docker Containerized

Multi-stage Docker builds with nginx reverse proxy, docker-compose for local dev, and production-ready container configuration.

8 Automated Tests

Jest + Supertest integration tests with in-memory MongoDB. Tests for auth, tasks, and health endpoints — all pass in CI.

CI/CD Pipeline

GitHub Actions workflows: CI runs tests on every push, CD builds and pushes Docker image to Docker Hub on merge.

Security First

Helmet.js headers, rate limiting, bcrypt password hashing, JWT authentication, and input validation with express-validator.

MongoDB Analytics

MongoDB aggregation pipelines for task statistics, priority breakdowns, overdue detection, and project progress tracking.
